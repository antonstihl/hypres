A presentation library

hatepres provides simple tooling to render semantic-enough HTML as a delightful visual aid for your presentation.

Why?

If you comfortably can code a presentation, why wouldn't you? :)

How to

  1. Create an .html file with your HTML content, divided into one <section/> per "slide".
  2. Add the head style tag <link rel="stylesheet" href="https://unpkg.com/hatepres@latest/styles.css" />
  3. Add the head script tag <script src="https://unpkg.com/hatepres@latest/script.js"></script>
  4. Add attribute hp-f to make elements focusable. This will hide them and reveal on focus.
  5. Done! ‚úîÔ∏è

Bonus

Tap into the classes/attributes used by hatepres to
style your elements as your presentation progresses.

In your HTML

  • [hp-f]
    • Bonus: Add a CSS selector as the hp-f value if you want to...
  • .p (perennial, used to escape [hp-f]'s default hiding)

In your CSS/JS

  • section.current - current section/slide
  • .current-focus - element is in focus
  • .focus-trace - element is or has been in focus

... make it an anchor that also focuses other elements. These anchored elements need to

  • have the attribute hp-fa ("focusable anchored")
  • match the CSS selector given in the anchor's hp-f value, e.g. by having a particular class.

Bonus II

There's also the global hatepres (or window.hatepres) object, with an API to add behaviour via JavaScript.

  • hatepres.focusNext()
  • hatepres.focusPrevious()
  • hatepres.goToNextSection()
  • hatepres.goToPreviousSection()
  • hatepres.next()

Now you can do unheard of things, like a button that triggers the next focus

Or why not a suspicious emoji that - when hovered - moves you to the next slide? Some JavaScript can make your presentations fun and interactive.

Philosophy

Some of the key characteristics of hatepres come from David JP Phillips TEDx talk, How to avoid death by PowerPoint.

    • One message per slide
    • Use contrast and size to direct attention
    • Don't write sentences
    • Dark bakground
    • Maximum of 6 items per slide
